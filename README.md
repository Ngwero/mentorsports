# Mentor Sports International Academy — Website

A modern, [Manchester United](https://www.manutd.com/en)-inspired website for [Mentor Sports International Academy](https://www.mentorsportsintug.com/).

## Pages

| Route | Description |
|---|---|
| `/` | Home — hero, news preview, program/team highlights |
| `/news` | Full news and articles |
| `/programs` | Holiday & youth programs |
| `/teams` | Age-group squads |
| `/trials` | Trial registration and process |
| `/tournaments` | Chipkizi Cup details |
| `/about` | Academy story, mission, core values |
| `/contact` | Contact form and details |

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS v4
- Lucide React icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Build for production:

```bash
npm run build
npm start
```

Deploy to Vercel, Netlify, or any static host. For WordPress replacement, you can export as static files or point your domain to the new deployment.

## Customization

- **Content**: Edit `src/data/content.ts` for all text, images, and links
- **Colors**: Adjust CSS variables in `src/app/globals.css`
- **Logo**: Replace the "MS" placeholder in `Header.tsx` and `Footer.tsx` with your academy crest image
